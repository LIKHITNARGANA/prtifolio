Introduction to C++

C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language. It adds object-oriented features and is widely used in software development, game programming, and systems programming.

C++ Variables

In C++, variables are used to store data. Each variable must be declared with a specific type before it is used. Here are common variable types in C++:

  • int - Integer type
  • float - Floating-point type
  • double - Double precision floating-point type
  • char - Character type
  • bool - Boolean type

Example:

        
          int age = 25;
          float salary = 55000.50;
          char grade = 'A';
          bool isStudent = true;

C++ Data Types

C++ supports several primitive data types, such as integers, floating-point numbers, characters, and booleans. You can also define your own custom data types using structures or classes.

Example:

        
          int num = 100;       // Integer
          double pi = 3.1415;  // Double precision
          char letter = 'A';   // Character
          bool flag = true;    // Boolean

C++ Control Structures

C++ provides several control structures such as conditional statements and loops to control the flow of execution.

Example of an if-else statement:

        
          int num = 10;
          if (num > 5) {
              cout << "Number is greater than 5" << endl;
          } else {
              cout << "Number is less than or equal to 5" << endl;
          }

C++ Functions

Functions are used to organize and reuse blocks of code. A function in C++ is defined using the return_type function_name() syntax.

Example:

        
          #include <iostream>
          using namespace std;

          int add(int a, int b) {
              return a + b;
          }

          int main() {
              int sum = add(5, 10);
              cout << "Sum is: " << sum << endl;
              return 0;
          }

C++ Loops

Loops allow you to repeat a block of code. C++ provides three main types of loops: for, while, and do-while loops.

Example of a for loop:

        
          for (int i = 0; i < 5; i++) {
              cout << i << endl;
          }

C++ Example Program

Here’s a simple C++ program that takes user input and displays a greeting:

        
          #include <iostream>
          using namespace std;

          int main() {
              string name;
              cout << "Enter your name: ";
              cin >> name;
              cout << "Hello, " << name << "!" << endl;
              return 0;
          }
