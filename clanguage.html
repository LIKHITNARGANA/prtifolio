Introduction to C

C is a general-purpose, procedural programming language. It was developed by Dennis Ritchie in the 1970s at Bell Labs. C is widely used for system programming, embedded programming, and application development.

Basic C Syntax

The basic structure of a C program includes:

  • #include - Includes libraries or header files.
  • int main() - The main function where the execution starts.
  • printf() - Used to display output on the screen.
  • return - Ends the execution of the program and returns a value.

Example of a simple C program:

        
          #include 

          int main() {
            printf("Hello, World!\n");
            return 0;
          }

Data Types in C

C supports several built-in data types to define variables:

  • int - Integer type.
  • float - Floating-point type.
  • double - Double-precision floating-point type.
  • char - Character type.

Example:

        
          int age = 25;
          float pi = 3.14;
          double largeNumber = 1.23456789;
          char grade = 'A';

Functions in C

Functions are blocks of code that perform specific tasks. Functions can take parameters and return values.

Example of a simple function:

        
          #include 

          void greet() {
            printf("Welcome to C Programming!\n");
          }

          int main() {
            greet();
            return 0;
          }

Loops in C

C has three primary types of loops:

  • for - A loop with a set number of iterations.
  • while - A loop that continues as long as the condition is true.
  • do-while - A loop that executes at least once, even if the condition is false.

Example:

        
          #include 

          int main() {
            int i;
            for(i = 1; i <= 5; i++) {
              printf("%d\n", i);
            }
            return 0;
          }

An array is a collection of elements of the same type stored in contiguous memory locations.
