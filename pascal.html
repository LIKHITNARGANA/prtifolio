Introduction to Pascal

Pascal is a high-level procedural programming language named after the mathematician Blaise Pascal. It was designed for teaching structured programming concepts and has a rich syntax, making it useful for learning programming fundamentals.

Basic Pascal Syntax

Pascal programs consist of a block structure that includes the program header, declarations, and statements. Below is an example of a simple Pascal program:

        
          program HelloWorld;
          begin
            writeln('Hello, world!');
          end.

Here, program is used to declare the program name, and begin and end mark the beginning and end of the program block.

Pascal Variables

Variables in Pascal are used to store data. Each variable must be declared with a specific type before it can be used. Common data types include integer, real, char, and boolean.

Example:

        
          var
            age: integer;
            name: string;
          begin
            age := 25;
            name := 'John';
            writeln('Age: ', age);
            writeln('Name: ', name);
          end.

Control Structures in Pascal

Pascal supports standard control structures like if statements, loops, and case statements for flow control.

If-Else Statement

        
          if age >= 18 then
            writeln('Adult')
          else
            writeln('Minor');

For Loop

        
          for i := 1 to 5 do
            writeln('Iteration: ', i);

While Loop

        
          while age < 30 do
            writeln('Still young!');

Pascal Procedures and Functions

Procedures and functions are blocks of code that can be reused throughout a program. Procedures do not return values, whereas functions return values.

Procedure Example

        
          procedure greet(name: string);
          begin
            writeln('Hello, ', name);
          end;
          begin
            greet('John');
          end.

Function Example

        
          function square(num: integer): integer;
          begin
            square := num * num;
          end;
          begin
            writeln('Square of 5: ', square(5));
          end.

Pascal Example

Here’s a complete Pascal program example:

        
          program SimpleExample;
          var
            num1, num2, sum: integer;
          begin
            writeln('Enter first number: ');
            readln(num1);
            writeln('Enter second number: ');
            readln(num2);
            sum := num1 + num2;
            writeln('Sum: ', sum);
          end.
