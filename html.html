Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML consists of various tags that define the structure and content of a webpage.

Basic HTML Tags

HTML uses a set of predefined tags to structure content on the web. Below are some common tags:

  • <html> - The root element of an HTML page.
  • <head> - Contains metadata about the document like title and links to stylesheets.
  • <body> - Contains the content of the HTML document.
  • <h1> to <h6> - Header tags for creating headings.
  • <p> - Paragraph tag for creating paragraphs.
  • <a> - Anchor tag for hyperlinks.
  • <img> - Image tag for embedding images.
  • <div> - Division tag used to group content.
  • <span> - Inline tag used to style a portion of text.

HTML Attributes

Attributes provide additional information about HTML elements. They are always written inside the opening tag and consist of a name and a value. For example:

  • href (in the <a> tag): Specifies the URL of the link.
  • src (in the <img> tag): Specifies the source of an image.
  • alt (in the <img> tag): Provides alternative text for an image.
  • id: A unique identifier for an element.
  • class: Defines the class of an element for CSS styling.

Example: <a href="https://www.example.com">Click here</a>

HTML Example

Here's a simple HTML example:

        
          <!DOCTYPE html>
          <html lang="en">
          <head>
            <meta charset="UTF-8">
            <title>Simple HTML Page</title>
          </head>
          <body>
            <h1>Hello World</h1>
            <p>This is a simple HTML page.</p>
          </body>
          </html>
