Introduction to Python

Python is a high-level, interpreted programming language. It was created by Guido van Rossum and first released in 1991. Python is widely used for web development, data science, artificial intelligence, and automation tasks.

Python Variables

In Python, a variable is used to store data. Variables are created by simply assigning a value to a name. There is no need to declare the type of variable beforehand.

Example:

        
          name = "John"
          age = 25
          is_student = True

Python Data Types

Python supports various data types:

  • int - Integer type.
  • float - Floating point type.
  • str - String type.
  • bool - Boolean type.
  • list - List type.
  • tuple - Tuple type.
  • dict - Dictionary type.

Example:

        
          num1 = 10          # Integer
          num2 = 20.5        # Float
          name = "Alice"     # String
          is_valid = True    # Boolean
          items = [1, 2, 3]  # List

Python Control Structures

Python has control structures like if-else statements, loops, and exception handling. These allow you to control the flow of your program.

Example of an if-else statement:

        
          num = 10
          if num > 5:
              print("Greater than 5")
          else:
              print("Less than or equal to 5")

Functions in Python

Functions allow you to group code into reusable blocks. You define a function using the def keyword.

Example of a function:

        
          def greet(name):
              return "Hello, " + name

          print(greet("Alice"))

Loops in Python

Loops help you repeat tasks. Python provides for and while loops.

Example of a for loop:

        
          for i in range(5):
              print(i)
