Introduction to Data Structures

Data structures are essential for organizing and managing data efficiently. They provide a way to store and retrieve data in a structured format. The choice of data structure directly affects the performance of algorithms.

Some commonly used data structures include arrays, linked lists, stacks, queues, trees, and graphs.

Arrays

An array is a collection of elements of the same type, stored in contiguous memory locations. Arrays allow random access to elements using indices.

Example of an array in C++:

        
          int arr[5] = {1, 2, 3, 4, 5};
          cout << "The third element is: " << arr[2] << endl;

Linked Lists

A linked list is a collection of nodes where each node contains data and a reference (or link) to the next node in the sequence. Unlike arrays, linked lists do not require contiguous memory locations.

Example of a simple linked list node in C++:

        
          struct Node {
              int data;
              Node* next;
          };
          Node* head = new Node();
          head->data = 10;
          head->next = nullptr;

Stacks

A stack is a linear data structure that follows the Last-In-First-Out (LIFO) principle. Elements are added and removed from the same end, called the top of the stack.

Example of using a stack in C++:

        
          stack s;
          s.push(10);
          s.push(20);
          s.pop();

Queues

A queue is a linear data structure that follows the First-In-First-Out (FIFO) principle. Elements are added from the rear and removed from the front.

Example of using a queue in C++:

        
          queue q;
          q.push(10);
          q.push(20);
          q.pop();

Trees

A tree is a hierarchical data structure consisting of nodes. Each node contains a value and references to child nodes. The top node is called the root.

Example of a simple binary tree node in C++:

        
          struct Node {
              int data;
              Node* left;
              Node* right;
          };
          Node* root = new Node();
          root->data = 10;

Graphs

A graph is a collection of nodes (vertices) and edges that connect pairs of nodes. Graphs can be directed or undirected and may contain cycles.
