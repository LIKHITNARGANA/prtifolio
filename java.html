Introduction to Java

Java is a widely used object-oriented programming language designed to be platform-independent and secure. It’s used in web development, mobile apps, enterprise software, and much more.

Basic Syntax

The basic structure of a Java program involves:

  • Class: Defines the blueprint of an object.
  • Main Method: The entry point of any Java program.
  • Statements: Each instruction ends with a semicolon.

Example:

        
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Java Data Types

Java supports several built-in data types. These include:

  • Primitive types: byte, short, int, long, float, double, char, boolean
  • Reference types: Objects, Arrays, Strings

Control Structures

Java provides control structures like:

  • If-else
  • Switch-case
  • For loop, while loop

Example of an if-else statement:

        
if (x > 10) {
    System.out.println("x is greater than 10");
} else {
    System.out.println("x is less than or equal to 10");
}

Example Java Program

Here’s a simple Java program that prints the sum of two numbers:

        
public class SimpleAddition {
    public static void main(String[] args) {
        int a = 5;
        int b = 10;
        int sum = a + b;
        System.out.println("Sum of a and b is: " + sum);
    }
}

Java Loops

Java supports different types of loops like:

  • For Loop
  • While Loop
  • Do-while Loop

Java Methods

Methods are blocks of code that perform a specific task and can be called by other parts of the program.

        
public class MethodsExample {
    public static void main(String[] args) {
        greet();
    }

    public static void greet() {
        System.out.println("Hello, Java!");
    }
}

Java Classes and Objects

Java is an object-oriented programming language. Classes are blueprints for creating objects, which are instances of classes.

        
public class Car {
    String model;
    int year;

    public Car(String model, int year) {
        this.model = model;
        this.year = year;
    }

    public void displayDetails() {
        System.out.println("Model: " + model + ", Year: " + year);
    }
}
